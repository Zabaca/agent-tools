# google-cli

Google Workspace CLI (Drive, Docs, Sheets) for Claude Code.

## Features

- Google Drive: list files, get metadata
- Google Docs: get, create, write (with markdown formatting support)
- Google Sheets: get info, read ranges (table/json/csv output)

## Setup

1. Install the plugin: `/plugin install google-cli@zabaca-agent-tools`
2. Authenticate: the skill will guide you through `login`

## Requirements

- Google OAuth credentials
- [Bun](https://bun.sh) runtime
